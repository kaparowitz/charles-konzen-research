## Calibration (NEW — add to memory)
Unlike the other Kirchenkreis Hofgeismar confirmation reels (219172 Heisebeck,
219742 Liebenau, 219673 Lamerden — all shuffled/duplicated 2014 re-filmings),
**219013 is cleanly chronologically ordered**, two written Seiten per image:
**Seite(left page) = 2 × image − 3914.** Verified: img 1985 = S56/57 (1850);
1984 = S54/55; 1983 = S52/53; 1982 = S50/51. Stale-canvas after a page-number
jump (shows previous leaf until a zoom +/- forces a redraw) — same fix as the
other Kassel registers; single-step prev/next arrows give clean redraws. Each
class lists **Knaben first, then Mädchen** ("Töchter"); running Nro resets each
year and a year-end Summa ("Summa h.a. N Confirmirte, x Knaben, y Töchter")
bounds every class. Columns: Nro | Name of confirmand | Names/standing of parents
| birth (year/month/day) | age | first-communion date | remarks.

## Result — NEGATIVE across the entire 1847–1855 window
Every confirmand in every class 1847–1855 read leaf-by-leaf (boys and girls,
~140+ entries) and checked for (a) forename Carl/Karl and (b) any Konze / *n-s-n /
*n-z-n surname.

- **No Konze / Kontze / Gansen / Gonsen / Konzen / *n-s-n surname appears
  anywhere** in the book's candidate window. The surname pool is entirely the
  recurring local Oberweser/Gieselwerder families: Marunde, Roll, Reuß, Nickel,
  Hennur, Vella, Vetta, Veitge, Schölla, Schölte, Grafmann, Grovenmann, Lindner,
  Otto, Lachmann, Daus, Sodte, Reitz, Greis, Müller, Masserschmidt, Deissel,
  Heinburg, Wasmuth, Hagen, Lorz, England, Tillmann, Göltz, Bumhof, Dürrbein,
  Sparr, Rabos, Pfennichs, Sülter, Mecklenburg, Witzel, Damm, Stölta, Pauen.
- **Exactly one Carl in the whole window:** **Carl Friedrich Wilhelm Marunde**,
  b. 21 March 1839, father Georg Wilhelm Marunde (Ackermann), mother née Simon —
  1853 class, entry 4 (img 1988 / S63). Local Marunde family, wrong surname,
  wrong birth year. Logged as an incidental Carl, not a candidate.
- **Prime cohorts decisive:** the 1849 class (14 boys, births clustering Jan
  1835 — entries 1/2/7 born 29 Jan / 20 Jan / 5 Jan 1835: Vetta, Veitge,
  Marunde) holds **no Carl**; the 1850 class (4 boys, births Dec 1835 – 1836:
  Schöll/Pfeil, Roll, Reuß, [Zahn]) holds **no Carl**. No boy born on/near
  26 Jan 1835 carries a Konze-type surname.

**Verdict:** Gieselwerder/Oberweser is now searched on BOTH the baptism
(Jan–Feb 1835, reg. 219016) AND the confirmation (1847–1855, reg. 219013)
tracks — negative on both. Parish remains ELIMINATED, now with confirmation-class
coverage matching the Eberschütz/Liebenau standard. Surname reads are by-eye
(not certified Kurrent) — hand to Peter Schräder for final confirmation, but the
given-name (Carl) and surname-skeleton (no Konze) results are reliable.
