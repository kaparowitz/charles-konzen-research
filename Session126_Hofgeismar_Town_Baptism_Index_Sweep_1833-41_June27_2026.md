# Session 126 — Hofgeismar Town Baptism Sweep (Altstädter 219265 + Neustädter 219364), 1833–1841
## Date: June 27, 2026

## Objective
Execute the **deferred baptism pass** for the **Hofgeismar town** parishes —
**Altstädter** and **Neustädter** Gemeinden — across the **1833–1841** window for:
(a) **Charles** — a boy named **Carl/Karl born 26 Jan 1835** later anglicised to
"Gannsen"; and (b) **any Konze / Kontze / Konzen / Konsen / Gansen / Gonsen /
Gonson / Gonnson / Conson / Jansen / *n·s·n surname** (child, parent, or godparent).

This closes the gap flagged in `germany/hofgeismar-research-log.html`: the town's
**baptism** track was "Not yet — deprioritised" (large urban classes, urban surname
pool). All prior Hofgeismar work (Sessions 63–64) searched the **marriage** registers
for the Konze × Hildebrand union and the **confirmation** reels — never the town
baptisms. Driven live in the logged-in Archion browser (subscription content).

## Register identification (corrects a long-standing ambiguity)
The "219253 / 219256 / 219295 / 219382" numbers in older notes are the Altstadt
**KB/Trauungen** and Neustadt **Trauungen** books — NOT baptisms. The actual town
**Taufen** registers are:
- **Altstädter — Taufen 1830–1843 = churchRegister 219265** (covers 1833–1841; next vol. Taufen 1843–1860).
- **Neustädter — Taufen 1830–1863 = churchRegister 219364** (covers 1833–1841).

Both carry **"Bem.: mit Register"** — i.e. each has an **alphabetical name index**
of every baptised child, spanning the register's full date range. The index is the
decisive instrument here: it lists every surname for the whole 1830–1843 / 1830–1863
span, so the K / G / J / C letter-sections answer the surname question for the entire
1833–1841 window in one pass.

## Method
For each register: located the alphabetical index at the end of the reel; read the
**K** (Konze/Konzen/Konsen/Kontze), **G** (Gansen/Gonsen/Gonson/Gonnson/Gannsen),
**I·J** (Jansen/Janson), and **C** (Conson/Contze) letter-sections by eye at high
zoom, both the *Männlich* (boys) and *Weiblich* (girls) columns. For Altstädter I
also read the **January 1835 birth-core leaves directly** for any boy born 26 Jan.
Stale-canvas quirk handled per house rule (zoom +/− to force a redraw after every
page-number jump).

- Baptisms begin **Bild 717 = Seite 2 (Aug 1830)**. **One Seite per opening ⇒ Seite = Bild − 715**
  (verified: Bild 820 = Seite 105 = Oct 1835).
- Alphabetical index at the **end** of the register: C ≈ Bild 977, D = 978, G = 981,
  H = 982, H/I·J = 983, **K = 984**, L = 985, M = 986, N/O = 988, Q/R = 991, U/V = 995.

  Kraft, Klingelhöfer — **NO Konze / Konzen / Konsen / Kontze**. (The only "Carls":

- **G (Bild 981), boys + girls:** Gueth/Guth, Grau, Guempel/Gümpel, Grebe, **Gante**,
  Goldbeck, Gülke, Giese, Groese/Grosse, George — **NO Gansen / Gonsen / Gonson /

## Bottom line
**Hofgeismar town baptisms 1833–1841 are NEGATIVE for Charles**, on **both** the
Altstädter (219265) and Neustädter (219364) parishes. Neither register's alphabetical
baptism index contains **any** Konze / Kontze / Konzen / Konsen / Gansen / Gonsen /
Gonson / Gonnson / Conson surname for its entire span. The only phonetically-adjacent
name — a local **"Janson"** family present in *both* parishes — is a distinct Hofgeismar
burgher line with **no son baptised Carl/Karl in the 1833–1841 window** (Altstadt: no
J-boy 1833–35 at all; Neustadt: Janson Johannes ~1839, Janson Carl Friedrich ~1860).
The direct Altstadt January-1835 read confirms **no boy born 26 Jan 1835**.

This **confirms the parish's existing ELIMINATED status on the baptism track** — prior
sessions had only ruled out Hofgeismar on the *marriage* (Konze × Hildebrand) and
*confirmation* tracks. The town's surname pool is, as the research log predicted,
urban and incompatible with Konze.
