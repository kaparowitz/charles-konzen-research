Eberschütz — Parish Research Log

51.613 N · 9.441 E · reg. 218887 · ~5.2 km NNE of Deisel · compiled 21 June 2026
ELIMINATED

Searched on two independent tracks, both negative. (1) Baptisms: Jan–Feb 1835 window read; the one illegitimate male — Heinrich Wilhelm Albore, b. 22 Jan 1835, mother an unmarried woman from Bischhausen (Kgr. Hannover) — verified at full zoom and judged not a match. (2) Confirmations (new, 23 Jun 2026; 1855–56 tail added 24 Jun 2026): the Eberschütz Konfirmationen register (reg. 218884) was read class-by-class for the full 1847–1856 span — every cohort in which a boy born 1833–1841 would be confirmed. The result is decisive: the prime 1849 class (births ~1835, the exact cohort for a boy born 26 Jan 1835) holds no Carl at all, and across the whole sweep only two boys named Carl appear — Carl Wilhelm Höbener (1854 class, b. Oct 1839, son of the shoemaker Johann Christoph Höbener) and Carl Scheele (1856 class, b. 1842, of an Eberschütz Scheele family) — both on non-matching surnames and both too young. No Konze / Kontze / Gansen / Gonsen / *n-s-n surname occurs anywhere in the register.

The 1856 class (Nro 324 ff.) was read into: its births are already 1842, so the 1841 birth-year is fully bracketed. Its only Carl is Carl Scheele (Nro 332, b. 1842, an Eberschütz Scheele) — wrong surname, born seven years after Charles. The single in-window birth among the 1856 entries is a girl (Heinze, b. 1840).

Register calibration (verified 23 Jun 2026): reg. 218884 runs two written Seiten per image at Seite (left) = 2 × image − 1731 — Bild 884 = Seite 37/38 = 1847, Bild 886 = S41/42 = 1849, Bild 893 = S55/56 = 1854, Bild 896 = S61/62 (1854 tail + 1855 class start), Bild 897 = S63/64 (1855 Summa + 1856 class start). Given names read reliably in the Kurrent; surname spellings are best by-eye readings (per project method) and the permalinks above carry the original images for a Kurrent specialist to confirm.

Remaining follow-ups

None. The 1855 class (births ~1841) and the start of the 1856 class were read leaf-by-leaf on 24 Jun 2026, closing the full 1833–1841 birth window at the leaf level — negative (see above). With the 1856 entries already running into 1842 births, no in-window cohort remains unread. This parish is searched on both the baptism and confirmation tracks — negative on both.
