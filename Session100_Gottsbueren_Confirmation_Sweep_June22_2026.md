# Session 100 — Gottsbüren Confirmation Sweep (with Archion permalinks / image numbers)

**Date:** 22 June 2026
**Goal:** Give Gottsbüren the confirmation-register read it had been missing. On the
search-coverage page Gottsbüren was the one Konze-pool parish "cleared via its
baptism/marriage/burial books, not a confirmation read," and its research log noted
"no per-image Archion permalinks were minted for this parish." This session does the
"any Carl, born 1833–1841, surname open" confirmation sweep and mints permalinks.

## Key structural finding

Gottsbüren keeps a **separate Konfirmationen 1831–1932 book** — Archion register
**219046** — distinct from the Taufen book (219049) that the project had been citing.
Confirmation section title page = Bild 1752; tabular classes follow.

The Konfirmationen section is **bound out of chronological order and contains duplicate
scans** (same pathology as the Hümme register 219532). Image number does not track
year or Seite. Verified live anchors: Bild 1774 = 1844, 1775 = 1845 (duplicated at
1788), 1777 = 1848-end/1849-start, 1778 = 1848 (full), 1780 = 1849 (boys 7–24),
1781 = 1849-end/tally, 1782 = 1850, 1784 = 1851, 1785 = 1852 (boys 14–19),
1776 = 1852 (boys 1–13, displaced into the 1846/47 slot). Each class lists the boys
(Knaben) numbered 1..N first, then "Mädchen"; only the boys were scanned.

## Verdict

Same **saturated negative** as every other swept parish: the forename *Carl* recurs
(~15 boys across classes 1848–1855) but **never once on a Konze-pool or *n-s-n* surname**,
and no Carl matches Charles. The most probative classes for a boy born 26 Jan 1835 —
**1849 and 1850** — were read in full and hold no Konze-type Carl (1850 holds no
primary Carl at all). With the residual pass adding 1853–1855 and confirming no 1846/47
class exists, **every confirmation class across the window (1844–1856) is now read.**
This upgrades Gottsbüren from "cleared on BMD books alone" to "confirmation register
fully swept, permalinks minted."

## Residual now closed — window swept end to end (1844–1856)

The second pass (22 Jun) pinned the 1853, 1854 and 1855 classes (table above), so every
confirmation class across Charles's whole 1833–1841 window has been read.

**1846 & 1847 appear not to exist.** Across a broad leaf-by-leaf sweep (~25 images,
Bild 1772–1797) the confirmed classes run 1844, 1845, then 1848, 1849 … 1856 — a clean
1845→1848 gap, no 1846/47 leaf surfacing. The section is non-monotonic and duplicated
(e.g. Bild 1773 = 1852 but 1774 = 1844; 1845 at both 1775 and 1788), so this is a
saturated hunt, not a missed jump. The gap parallels the register's earlier 1831→1834
gap and most likely means no class was held those two years (vacancy / biennial). To be
confirmed on-site, but with 1844–45 and 1848–56 all read, no boy born 1831–1842 with a
Konze-type surname exists in this book.

Reliability caveat unchanged: Carl (vs. Conrad/Johann) is dependable in Kurrent; exact
birth-year digits and surname spellings are best readings, not certified transcriptions.
