## Register structure — important finding

Kelze has **no separate Konfirmationen book**. Its baptisms, confirmations,
marriages and burials are all in **one combined volume**: Archion
`churchRegister/219634` = **"Kelze KB 1830–1971"**, Kirchenkreis Hofgeismar,
Landeskirchliches Archiv der EKKW (834 images).

The filming title card (**Bild 364**) gives the whole section map:
- `*` (Taufen) 1830–1970 → **Seite 2–166**
- **`Konf` (Konfirmationen) 1831–1970 → Seite 1–119**
- `oo` (Trauungen) 1830–1932 → Seite 1–64
- `+` (Todten) 1830–1971 → Seite 1–112
- Register (index) after each section.

**Confirmation calibration (verified this session):** the Konfirmationen are
WELL-ORDERED (no shuffle/duplication, unlike 219532/219046/219172). Each image =
one two-page opening; each Seite carries its own "Seite N — Jahr 18XX" header and
column set (Nro · Namen der Konfirmierten · Namen u. Stand der Eltern ·
Geburtsjahr u. Tag · Alter · Datum der Confirmation · Bemerkungen). Year-end
tally lines read "Sa. der Confirmierten in 18XX: N. x Knaben, y Mädchen".

## Result — NEGATIVE

- **No Konze / Kontze / Gansen / Ganssen / Gonsen / Gonson / Konzen / Konsen /
  Jansen / Hansen surname appears in any class 1847–1855.** Every confirmand
  bears a tight local set of surnames — overwhelmingly **French-Huguenot-colony
  names** (Bonnet, Benoit, Morell, Monard, Morin, Croll, Briede; maiden names
  Seguin, Paschal, Ruddenklau), plus Homburg, Weidemann, Stalknecht, Siebert,
  Finis/Tinnes, Jäger, Martin, Schöning, Rudolph. Kelze was a Huguenot colony, so
  a German Diemel-valley "Konze" would be conspicuous — and it is simply absent.
- **Exactly one Carl in the whole window: Carl Wilhelm Homburg, b. 4 Dec 1835**
  (1850 class, son of a Homburg Tagelöhner/Schullehrer family) — wrong surname
  (-burg, not a Konze/\*n-s-n) and born **December** 1835, not Charles's
  **26 January** 1835. Ruled out.
- **Decisively, the 1849 class — the exact cohort for a boy born 26 Jan 1835 —
  holds NO Carl at all** (4 Knaben: F. W. Homburg, Ludwig Finis, + the two on
  Seite 17; 1 Mädchen). The adjacent 1850 class's only Carl is the Homburg above.

This matches every other Kirchenkreis Hofgeismar parish swept (Heisebeck,
Liebenau, Lamerden, Eberschütz, Schöneberg, Gottsbüren, Sielen, Hümme,
Helmarshausen, Gudensberg): the forename Carl recurs on local surnames, never on
a Konze-type one. Kelze's prior **ELIMINATED** status (set on the Jan–Feb 1835
baptisms) now stands on BOTH the baptism and the full confirmation tracks.
