## ⏳ DEATH — register located, entry not yet extracted (Nürnberg St. Sebald 1808)

It carries **running death numbers reaching ~100+ per year** and dense ~30+ entries per page, so the cert's "p. 9 / no. 35" does not map one-to-one to the viewer's pages, and the 1807 vs 1808 section boundary needs reading out. The robust path is to find the entry **by date (29 March 1808) + name** in the 1808 section, not by the cert's page/number.
- **SEARCH RESULT (June 13) — NOT FOUND in this register.** Worked the 1808 section by date+name (the reliable method). The register is a microfiche scan (fiche header: "NUERNBERG ST. SEBALD · BEERDIGUNGEN 1807-1810 (+REG.) · SIGN. 5-53 · BL. 1-21"). Read **every burial entry across the 8 March – mid April 1808 window** (running nos. 21–58, pages ~12–13): Bröselin, Vetterlein, Ernst, Geräck, **Häußlein (entry 35, ~29 März — a woman, NOT Rabus)**, Stein, Fürst, Schütz, Königin/Pfann, Stettler, Heinrich, Geller, Männlein, Vock, Walther, Zahn, Quell, Aggott, Wibel, Stöllin, von Eichholtz, Vögner, Langhans … **no Johann Adam Rabus.** This book holds ~16–24 entries per page, so the certificate's "p. 9 = entry 35" (≈4 entries/page) cannot be this register; page 9 here is early-1808 (January), which can't hold a 29 March death. ## Technique notes (for future sessions)
- Archion viewer serves only a ~776px thumbnail until you **raise the zoom slider** (click the "+" control ~10×) — then it fetches a high-res layer readable for Kurrent. Use the screenshot `zoom` action on top of that.
- **Permalink:** viewer toolbar chain icon → check "Zoomstufe & Bildbereich einschließen" → Erstellen → gives a stable `archion.de/p/<hash>/` link.
- **Download:** viewer toolbar download icon → Hoch-/Querformat → Herunterladen → saves a 1-page PDF of the current view to the user's Downloads.
- **FS Memories upload via browser:** open person → Memories → ADD MEMORIES → Upload (do NOT click the visible button blindly; locate the hidden `input[type=file]` and upload the file path directly). "Use file names as titles" is on by default, so name the file descriptively first. Then fill Description (citation + transcription + translation), Place, Date, set **Make Public**, Done.
- The user's pre-downloaded Archion PDFs can be uploaded directly from the session uploads/outputs folders.
